# learning-javascript
Game creation with JS and PIXI

Run http-server with "npm start"

Push to Heroku
git push heroku master

Push to GitHub
git push

Start 1 web node at Heroku
heroku ps:scale web=1

Stop every Heroku web node
heroku ps:scale web=0

View Heroku logs
heroku logs --tail

Open Heroku live site
heroku open
